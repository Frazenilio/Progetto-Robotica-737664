# Progetto-Robotica-737764
Progetto di Robotica per 737664 - Micromouse

## Come usare il micromouse
1. Scaricare micromouse dalla repository originale (sezione Download del README): https://github.com/mackorone/mms-python/tree/master 
2. Avviare mms.exe
3. Aggiungere un nuovo profilo cliccando + vicino alla sezione "Mouse"
4. Compilare i campi sulla base del modello. Per eseguire questo progetto, è necessario Python
5. Alla sezione "" inserire il comando per eseguire main python. Ad esempio: "
